# Financial Document Annotation Framework

## Overview

This project implements a structured annotation framework for financial documents using synthetic datasets and standardized entity extraction methodologies.

## Features

- Financial document classification
- Entity extraction schema
- OCR-aware annotation simulation
- Inter-annotator agreement analysis
- Dataset statistics generation
- Edge case documentation

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset

Total Documents: 200

## Entity Types

22 Financial Entity Types

## Outputs

- annotation_schema.json
- annotations.jsonl
- statistics.json
- edge_cases.json
- final_report.txt
